# arien.dev

Personal portfolio website of Aryan Jain.

## Overview

- **Stack:** Vite, React, TypeScript, Tailwind CSS, shadcn/ui.
- **Features:** Dark mode (Zen theme), responsive design, interactive components.

## Development

```bash
# Install dependencies
bun install

# Start dev server
bun run dev
```

## Deployment

Built with Vite, deployable to Vercel, Netlify, or similar platforms.
